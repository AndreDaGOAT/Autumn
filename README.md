+# Autumn Memories
+
+A simple GitHub Pages-ready site for sharing childhood memories and pictures on a gentle rotation.
+
+## How it works
+
+- `index.html` provides the page structure and gallery controls.
+- `styles.css` creates a warm autumn-inspired layout that works on desktop and mobile.
+- `script.js` stores the memory slides and rotates through them every five seconds.
+
+## Add your own childhood photos
+
+Open `script.js` and replace each `image`, `title`, `date`, `caption`, and `alt` value in the `memories` array with your own family photos and stories. Photos can be hosted in the repository, GitHub issue uploads, or another trusted image host.
+
+## Publish with GitHub Pages
+
+1. Push these files to the repository.
+2. Open the repository settings on GitHub.
+3. Go to **Pages**.
+4. Choose the current branch and the repository root as the source.
+5. Save the settings and open the published Pages URL when GitHub finishes deploying.
